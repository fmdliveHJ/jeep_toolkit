## 구현 기능

- react
- ❗️redex-toolkit
- 스와이퍼 슬라이드
- 바닐라 스크립트 슬라이드
- fetch이용한 json 데이터 호출
- flickr, youtube 재생목록, api 호출
- 자바스크립트 스크롤 모션
